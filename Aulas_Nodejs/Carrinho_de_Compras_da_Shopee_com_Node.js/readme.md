# Carrinho de Compras da Shopee

## 💼 Sobre o projeto

Este repositório foi criado para ser o projeto 'Carrinho de Compras da Shopee com Node.js' da Formação Node.js da DIO.

## ✅ Objetivo

Criar um carrinho de compras baseado no carrinho de compras da shopee, aonde o carrinho armazene itens e faça o cálculo total e de sub-itens automaticamente.
